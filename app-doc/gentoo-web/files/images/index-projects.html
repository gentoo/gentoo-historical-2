Gentoo Linux Projects

The dynfw firewall scripts are a collection of netfilter-based (iptables) firewall scripts designed to allow for an immediate, measured response against pending security threats. These bash scripts have been designed to work with nearly any netfilter-based firewall (including both "allow by default" and "deny by default" configurations).

The XML projects page contains the complete guide XSLT engine that is used to create this entire site, including the main pages and online documentation. On this page, you can find a tarball that contains the complete sources that were used to generate the latest version of the http://www.gentoo.org Web site -- the one you are viewing now.
